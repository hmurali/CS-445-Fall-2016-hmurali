How to build each program:
1. run the MAKEFILE 'makefile.mak'
2. running the file from step 1 will compile the programs included in this zip file when executed.
3. IMPORTANT: run 'make' command to compile and build java source files.

How to run each program:
1. Run Main.java from default package, button package, and lightbulb package in TableLamp project.
2. Run TableLamp.java, ElectricPowerSwitch.java, Fan.java, and Lightbulb.java.
3. JUnit tests are included in ElectricPowerSwitchTest.java; run this file with JUnit (the testing tool I used to run
my unit tests for this assignment).
